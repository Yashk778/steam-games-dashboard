# 🎮 Steam Games Dashboard

An end-to-end data science project built on 2000+ Steam games.

## Project Steps
- Web scraping Steam store using BeautifulSoup
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Interactive dashboard built with Streamlit & Plotly

## Features
- Genre & sentiment analysis
- Release trends over years and months
- Top developers & publishers
- Pricing insights
- Popularity analysis

## Tech Stack
Python, BeautifulSoup, Pandas, Streamlit, Plotly
